1.**概要**

このgithubは、2026年度秋学期にて早稲田大学社会科学部「政治過程論2」で提出した「」のリプリケーションファイルを目的に整備されたものです。
<br>早稲田moodle上で提出を試みたところ、最大ファイルサイズ50MB、最大ファイル数が20件と書かれていたので、再現性の確保及び結果の解釈として用いたデータおよび資料を適切にアップロードできませんでした。
<br>GoogleDriveなどのオンライン上での提出も検討いたしましたが、今回はGitHub上での共有とさせていただきます。
<br>どうぞよろしくお願いたします。

2.**各フォルダの説明**

- code

  - 結果を再現するためのコードがあります。
  - 1.トピック数計算は2時間ほどかかるので、実行を推奨しない。

- data

  - 分析に際し用いたデータ及びstm_modelや、メタ情報などがあります。

- result

  - いわゆるAppendixです。
  - 本文中で解釈を行った際に用いた資料があります。
  - 資料を確認の際は、こちらに入っているファイルを適宜ご覧ください。

- packages

  - 再現するために中国語を分ち書きするためのjiebaRD及びjiebaRパッケージのgzファイルを配置しました。

- repulication

  - codeを実行した際に得られるものです。

  - これらのうち、本文中で言及したものはresultフォルダにあります。
